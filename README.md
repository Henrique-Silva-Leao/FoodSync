# 🚚 TruckSys Web

> **Sistema Integrado de Gerenciamento para Food Trucks e Lanchonetes**

## 📖 Sobre o Projeto

Este projeto é um aplicativo web desenvolvido como Trabalho de Graduação (TG) e requisito da disciplina de Engenharia de Software III. O objetivo é prover suporte tecnológico acessível por meio de Tecnologias Digitais de Informação e Comunicação (TDIC) para a gestão operacional de microempreendimentos gastronômicos.

O sistema centraliza, em uma interface única e responsiva, diversas operações críticas:
* Registro de pedidos na mesa via dispositivos móveis.
* Atualização de cardápio dinâmico.
* Controle de estoque com baixa automática de insumos em tempo real.

**Foco Principal:** Otimizar as operações diárias e reduzir falhas e desperdícios.

---

## 🚀 Funcionalidades Principais

* **📱 Atendimento Mobile:** Painel de Ponto de Venda (PDV) otimizado para celulares e tablets, permitindo que o garçom registre pedidos diretamente na mesa ou no balcão.
* **⚙️ Processamento Paralelo:** Envio simultâneo da ordem de preparo para a área de produção (Cozinha/KDS) e baixa automática no inventário de insumos (Just-In-Time) com base na ficha técnica do prato.
* **📋 Cardápio Dinâmico:** Atualização e gestão de produtos, preços, observações de clientes e disponibilidade em tempo real.
* **📦 Gestão de Estoque Perpétuo:** Controle rigoroso de ingredientes para evitar desperdícios, compras desnecessárias e rupturas na produção.

---

## 💻 Tecnologias Utilizadas

O projeto utiliza uma stack tecnológica moderna, robusta e focada em alta performance e segurança:

* **Front-end:** HTML e framework Razor Blazor (Single-Page Applications interativas no lado do cliente).
* **Back-end:** C# (Ecossistema .NET) para construção das regras de negócio e APIs.
* **Banco de Dados:** PostgreSQL (Relacional) para garantir a integridade da baixa de estoque.
